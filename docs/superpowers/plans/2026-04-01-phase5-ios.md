# VideoFeed Phase 5 — iOS Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the full iOS app using SwiftUI, consuming all KMP shared ViewModels via SKIE for clean Swift/coroutine interop. Covers Xcode project setup, iOS-specific platform implementations (TokenStorage via NSUserDefaults), and all three feature screens: auth (login + register), feed (TabView paging + AVPlayer + prefetch), and upload (PhotosPicker + progress).

**Architecture:** SKIE wraps the KMP framework so `StateFlow<T>` becomes an `AsyncSequence` in Swift and `suspend` functions become `async`. SwiftUI views observe state via `.task { for await state in viewModel.state { ... } }`. iOS ViewModels are created by iosMain factory functions that accept a `CoroutineScope` (Swift side creates and cancels it per view lifecycle). Koin is initialised once at app startup via a shared `initKoin()` KMP function.

**Tech Stack:** Swift 5.9+, SwiftUI, AVKit/AVFoundation, PhotosUI, SKIE 0.9.x, Kotlin/Native XCFramework, KMP Koin

```kotlin
package br.gohan.videofeed.core.data

import platform.Foundation.NSUserDefaults

class TokenStorageNSUserDefaults : TokenStorage {
    private val defaults = NSUserDefaults.standardUserDefaults
    private val key = "videofeed_jwt_token"

    override fun saveToken(token: String) {
        defaults.setObject(token, forKey = key)
    }

    override fun getToken(): String? {
        return defaults.stringForKey(key)
    }

    override fun clearToken() {
        defaults.removeObjectForKey(key)
    }
}
```

```kotlin
package br.gohan.videofeed.ioskoin

import br.gohan.videofeed.core.data.coreDataModuleIOS
import br.gohan.videofeed.feature.auth.data.authDataModule
import br.gohan.videofeed.feature.auth.domain.AuthRemoteDataSource
import br.gohan.videofeed.feature.auth.presentation.LoginViewModel
import br.gohan.videofeed.feature.auth.presentation.RegisterViewModel
import br.gohan.videofeed.feature.feed.data.feedDataModule
import br.gohan.videofeed.feature.feed.domain.VideoRemoteDataSource
import br.gohan.videofeed.feature.feed.presentation.FeedViewModel
import br.gohan.videofeed.feature.upload.data.uploadDataModule
import br.gohan.videofeed.feature.upload.domain.R2UploadDataSource
import br.gohan.videofeed.feature.upload.domain.UploadRemoteDataSource
import br.gohan.videofeed.feature.upload.presentation.UploadViewModel
import kotlinx.coroutines.CoroutineScope
import org.koin.core.context.startKoin
import org.koin.core.parameter.parametersOf

fun initKoin(baseUrl: String) {
    startKoin {
        properties(mapOf("baseUrl" to baseUrl))
        modules(
            coreDataModuleIOS,
            authDataModule,
            feedDataModule,
            uploadDataModule
        )
    }
}

object IOSViewModelFactory {
    fun loginViewModel(scope: CoroutineScope): LoginViewModel {
        val koin = org.koin.core.context.GlobalContext.get()
        return LoginViewModel(
            authDataSource = koin.get(),
            tokenStorage = koin.get(),
            coroutineScope = scope
        )
    }

    fun registerViewModel(scope: CoroutineScope): RegisterViewModel {
        val koin = org.koin.core.context.GlobalContext.get()
        return RegisterViewModel(
            authDataSource = koin.get(),
            tokenStorage = koin.get(),
            coroutineScope = scope
        )
    }

    fun feedViewModel(scope: CoroutineScope): FeedViewModel {
        val koin = org.koin.core.context.GlobalContext.get()
        return FeedViewModel(
            videoDataSource = koin.get(),
            coroutineScope = scope
        )
    }

    fun uploadViewModel(scope: CoroutineScope): UploadViewModel {
        val koin = org.koin.core.context.GlobalContext.get()
        return UploadViewModel(
            remoteDataSource = koin.get(),
            r2DataSource = koin.get(),
            coroutineScope = scope
        )
    }
}
```

> **Note:** The presentation Koin modules (`authPresentationModule`, `feedPresentationModule`, `uploadPresentationModule`) use `viewModelOf` which is Android-specific. iOS bypasses them and uses `IOSViewModelFactory` directly. Do NOT include `*PresentationModule` in `initKoin()` on iOS.

- [ ] **Step 3: Create `ViewModelHolder.swift`**

This generic `ObservableObject` manages the `CoroutineScope` lifecycle. When the holder is deallocated (view disappears), the scope is cancelled, stopping all coroutines in the ViewModel.

```swift
import Foundation
import VideoFeed  // KMP framework
import Combine

/// Manages a KMP CoroutineScope tied to this object's lifetime.
/// Subclass and create your ViewModel in the initialiser.
class ViewModelHolder<T: AnyObject>: ObservableObject {
    let scope: Kotlinx_coroutines_coreCoroutineScope
    let viewModel: T

    init(factory: (Kotlinx_coroutines_coreCoroutineScope) -> T) {
        // Main dispatcher scope — matches Android's viewModelScope
        let dispatcher = Kotlinx_coroutines_coreMainDispatcherKt.getMain()
        scope = MainScopeKt.MainScope()
        viewModel = factory(scope)
    }

    deinit {
        scope.cancel(cause: nil)
    }
}
```

- [ ] **Step 2: Create `VideoPreloader.swift`**

Pre-loads the `playable` and `duration` properties of the next 2 video assets so they are ready before the user swipes to them.

```swift
import Foundation
import AVFoundation

actor VideoPreloader {
    private var loadedAssets: [String: AVAsset] = [:]

    func preload(urls: [URL]) async {
        for url in urls {
            let urlString = url.absoluteString
            guard loadedAssets[urlString] == nil else { continue }
            let asset = AVAsset(url: url)
            do {
                _ = try await asset.load(.isPlayable)
                _ = try await asset.load(.duration)
                loadedAssets[urlString] = asset
            } catch {
                // Prefetch failure is non-fatal — video will load on demand
            }
        }
    }

    func asset(for url: URL) -> AVAsset? {
        loadedAssets[url.absoluteString]
    }
}
```

- [ ] **Step 3: Create `FeedView.swift`**

```swift
import SwiftUI
import AVFoundation
import VideoFeed

private class FeedHolder: ViewModelHolder<FeedViewModel> {
    init() {
        super.init { scope in
            IOSViewModelFactory.shared.feedViewModel(scope: scope)
        }
    }
}

struct FeedView: View {
    @StateObject private var holder = FeedHolder()
    @State private var state: FeedState = FeedState(
        videos: [], isLoading: false, error: nil, currentIndex: 0
    )
    @State private var currentIndex = 0
    @State private var players: [String: AVPlayer] = [:]
    private let preloader = VideoPreloader()

    let onNavigateToLogin: () -> Void
    let onNavigateToUpload: () -> Void

    private var vm: FeedViewModel { holder.viewModel }

    var body: some View {
        GeometryReader { geo in
            ZStack(alignment: .topTrailing) {
                if state.videos.isEmpty && state.isLoading {
                    ProgressView()
                        .frame(maxWidth: .infinity, maxHeight: .infinity)
                } else {
                    TabView(selection: $currentIndex) {
                        ForEach(Array(state.videos.enumerated()), id: \.element.id) { index, video in
                            VideoItemView(
                                video: video,
                                player: player(for: video),
                                isVisible: index == currentIndex,
                                size: geo.size
                            )
                            .tag(index)
                        }
                    }
                    .tabViewStyle(.page(indexDisplayMode: .never))
                    .ignoresSafeArea()
                    .onChange(of: currentIndex) { _, newIndex in
                        vm.onAction(action: FeedAction.OnVideoVisible(index: Int32(newIndex)))
                        prefetchNext(from: newIndex)
                    }
                }

                // Upload button (top-right)
                Button(action: onNavigateToUpload) {
                    Image(systemName: "plus.circle.fill")
                        .font(.title)
                        .padding()
                }
                .foregroundColor(.white)
            }
        }
        .task {
            for await newState in vm.state {
                state = newState
            }
        }
        .task {
            for await event in vm.events {
                if event is FeedEvent.NavigateToLogin {
                    onNavigateToLogin()
                }
            }
        }
        .onAppear {
            vm.onAction(action: FeedAction.OnRefresh())
        }
    }

    private func player(for video: VideoUi) -> AVPlayer {
        if let existing = players[video.id] { return existing }
        let url: URL
        if let cached = preloader.asset(for: URL(string: video.cdnUrl)!) {
            url = URL(string: video.cdnUrl)!
            let item = AVPlayerItem(asset: cached)
            let player = AVPlayer(playerItem: item)
            players[video.id] = player
            return player
        } else {
            let player = AVPlayer(url: URL(string: video.cdnUrl)!)
            players[video.id] = player
            return player
        }
    }

    private func prefetchNext(from index: Int) {
        let nextUrls = (1...2).compactMap { offset -> URL? in
            let nextIndex = index + offset
            guard nextIndex < state.videos.count else { return nil }
            return URL(string: state.videos[nextIndex].cdnUrl)
        }
        Task { await preloader.preload(urls: nextUrls) }
    }
}

struct VideoItemView: View {
    let video: VideoUi
    let player: AVPlayer
    let isVisible: Bool
    let size: CGSize

    var body: some View {
        ZStack(alignment: .bottomLeading) {
            VideoPlayerView(player: player)
                .frame(width: size.width, height: size.height)
                .onAppear { if isVisible { player.play() } }
                .onDisappear { player.pause() }
                .onChange(of: isVisible) { _, visible in
                    if visible { player.play() } else { player.pause() }
                }

            // Info overlay
            VStack(alignment: .leading, spacing: 4) {
                Text(video.title)
                    .font(.headline)
                    .foregroundColor(.white)
                Text(video.uploaderName)
                    .font(.subheadline)
                    .foregroundColor(.white.opacity(0.8))
            }
            .padding()
            .padding(.bottom, 40)
        }
    }
}
```

```swift
import PhotosUI
import VideoFeed

private class UploadHolder: ViewModelHolder<UploadViewModel> {
    init() {
        super.init { scope in
            IOSViewModelFactory.shared.uploadViewModel(scope: scope)
        }
    }
}

struct UploadView: View {
    @StateObject private var holder = UploadHolder()
    @State private var state: UploadState = UploadState(
        title: "", selectedFilename: nil, status: UploadStatusIdle()
    )
    @State private var pickerItem: PhotosPickerItem? = nil
    let onSuccess: () -> Void

    private var vm: UploadViewModel { holder.viewModel }

    private var isLoading: Bool {
        state.status is UploadStatusPresigning
        || state.status is UploadStatusUploading
        || state.status is UploadStatusFinalizing
    }

    private var canSubmit: Bool {
        !isLoading && state.selectedFilename != nil && !state.title.isEmpty
    }

    var body: some View {
        VStack(spacing: 24) {
            Text("Upload Video")
                .font(.title)
                .bold()

            PhotosPicker(
                selection: $pickerItem,
                matching: .videos,
                photoLibrary: .shared()
            ) {
                Label(
                    state.selectedFilename ?? "Select Video",
                    systemImage: "video.badge.plus"
                )
                .frame(maxWidth: .infinity)
            }
            .buttonStyle(.bordered)
            .disabled(isLoading)
            .onChange(of: pickerItem) { _, item in
                guard let item else { return }
                Task { await loadVideo(from: item) }
            }

            TextField("Title", text: Binding(
                get: { state.title },
                set: { vm.onAction(action: UploadAction.OnTitleChange(title: $0)) }
            ))
            .textFieldStyle(.roundedBorder)
            .disabled(isLoading)

            // Status indicator
            switch state.status {
            case let uploading as UploadStatusUploading:
                VStack(spacing: 8) {
                    ProgressView(value: Double(uploading.progress))
                    Text("Uploading… \(Int(uploading.progress * 100))%")
                        .font(.caption)
                }
            case is UploadStatusPresigning:
                HStack {
                    ProgressView()
                    Text("Preparing upload…")
                }
            case is UploadStatusFinalizing:
                HStack {
                    ProgressView()
                    Text("Saving…")
                }
            case is UploadStatusDone:
                Text("Upload complete!")
                    .foregroundColor(.green)
            case let error as UploadStatusError:
                Text(error.message)
                    .foregroundColor(.red)
                    .font(.caption)
            default:
                EmptyView()
            }

            Button(action: { vm.onAction(action: UploadAction.OnSubmit()) }) {
                Text("Upload")
                    .frame(maxWidth: .infinity)
            }
            .buttonStyle(.borderedProminent)
            .disabled(!canSubmit)

            Spacer()
        }
        .padding()
        .task {
            for await newState in vm.state {
                state = newState
            }
        }
        .task {
            for await event in vm.events {
                if event is UploadEvent.NavigateToFeed {
                    onSuccess()
                }
            }
        }
    }

    private func loadVideo(from item: PhotosPickerItem) async {
        guard let url = try? await item.loadTransferable(type: URL.self) else { return }
        guard let data = try? Data(contentsOf: url) else { return }
        let filename = url.lastPathComponent
        let mimeType = "video/mp4"
        let bytes = [UInt8](data)
        let kotlinBytes = KotlinByteArray(size: Int32(bytes.count))
        for (i, b) in bytes.enumerated() {
            kotlinBytes.set(index: Int32(i), value: b.toInt8())
        }
        vm.onAction(action: UploadAction.OnFileSelected(
            bytes: kotlinBytes,
            filename: filename,
            mimeType: mimeType
        ))
    }
}
```

> **Note on `ByteArray` conversion:** KMP exposes `KotlinByteArray` in Swift. The conversion loop above is correct but can be slow for large files. For production, use a `@Sendable` background task and show a loading indicator during file read. For this PoC it is sufficient.
